# jupyterlab_cjson

A JupyterLab and Jupyter Notebook extension for rendering CJSON

## Prerequisites

* JupyterLab ^0.20.0 and/or Notebook >=4.3.0

## Install

```bash
jupyter labextension install jupyterlab_cjson
```

## Development

```bash
pip install -e .
cd labextension
jupyter labextension link .
```
